# Machine Learning Journey

A structured, hands-on learning repository covering core machine learning algorithms from scratch using Python and Jupyter Notebook. Each folder focuses on one algorithm with implementation, explanation, and examples.

## Topics Covered

| # | Topic | What's Covered |
|---|-------|----------------|
| 01 | Linear Regression | Simple & multiple regression, cost function, gradient descent |
| 02 | Logistic Regression | Binary classification, sigmoid function, decision boundary |
| 03 | Logistic Regression (Multiclass) | One-vs-Rest strategy, multi-class classification |
| 04 | One Hot Encoding | Handling categorical variables for ML models |
| 05 | Train & Test Split | Avoiding data leakage, evaluating model generalisation |
| 06 | Decision Tree | Entropy, information gain, tree visualisation |
| 07 | Random Forest | Ensemble learning, bagging, feature importance |
| 08 | Support Vector Machine | Hyperplane, margin, kernel trick |

## Tech Stack

- **Language:** Python 3
- **Libraries:** Scikit-learn, Pandas, NumPy, Matplotlib
- **Environment:** Jupyter Notebook

## How to Run

```bash
git clone https://github.com/Adithyan-7/machine-learning-journey.git
cd machine-learning-journey
pip install scikit-learn pandas numpy matplotlib jupyter
jupyter notebook
```

Open any numbered folder and run the `.ipynb` file inside.
